# webcrawler

## wiki crawler 
### Find a path using Breadth-first search from one Wikipedia article to another by only using links

#### On Linux
```
python3 wiki_path.py <from> <to>
```

#### On Windows 
```
py wiki_path.py <from> <to>
```


### Requirements 
* python3 (v3.7)
* requests_html (pip)
